# switchEtherNetwork

The `switchEtherNetwork` function expects a chainId in HEX base.
You can parse a decimal id using the <Link href="/usemetamask/parse">parse nano-module</Link>.

```jsx
import { switchEtherNetwork } from "@forta/usemetamask"

const RIKEBY_ID = "0x4"

function SwitchToRinkeby() {
  return (
    <Button onClick={() => switchEtherNetwork(RIKEBY_ID)}>
      Switch Network
    </Button>
  )
}
```
